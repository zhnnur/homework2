1. DRY (Қайталама) — Logger.java, DatabaseConfig.java
Не істелді: Қайталанатын код бір ортақ әдіске жиналды. Конфигурациялар жеке константаға шығарылды.

Нәтиже: Кодты өзгерту оңай, бір жерде түзетсеңіз жеткілікті.

2. KISS (Қарапайым ету) — NumberProcessor.java, Calculator.java
Не істелді: Көп қабатты if шарттары «guard clauses» әдісімен жойылды. try-catch блогы қарапайым if тексеруімен алмастырылды.

Нәтиже: Код түсінікті, логика «тегіс» және тез жұмыс істейді.

3. YAGNI (Артық кодты жой) — User.java, ReportGenerator.java
Не істелді: Болашақта керек болып қалар деген артық әдістер (Excel, HTML есептері) және басы артық параметрлер жойылды.

Нәтиже: Тек қазіргі уақытта қажетті функциялар ғана қалды, жоба «қоқыстан» тазартылды.
